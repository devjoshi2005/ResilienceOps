# ResilienceOps - A Cloud Native Incident Detection Engine

Github Project demonstrating the working of a cloud native incident detection platform where multiple resources like AWS S3 (Simple Storage Bucket) , IAM (Identity Access Management) roles and EKS (Elastic Kubernetes Cluster) are scanned everyday for vulnerabilties (by guardduty) and misconfigurations (by cloudtrail)
